Importing Data

Data should be imported through an excel file that checks all the necessary requirements (See Excel File Criteria) New data should be imported regurarly, in the beginning of every month. If the data is not imported on time, a reminder will occure.

Excel File Criteria


As mentioned in the Importing data section, the excel file that contains data should comply to certain criteria.

Please ensure that your Excel file complies with the following rules:

  • Column Names: Column names in Excel must match exactly the column names in the template provided in the end of this section. This includes upper/lower case.

    Avoid using special characters in column names, such as @, #, !, etc.

    No spaces in column names, replace with “_”.

  • Data Formatting: If you have date columns for example, make sure that the date values in Excel are properly formatted.

  • Number of Columns: The Excel file should have the same number of columns as the template

  • Column Order: Column order is generally not important if the names match.

  • File Format: Ensure that the Excel file is in a .xls or .xlsx format. Other file formats will not be supported.

  • No Merged Cells: Merged cells are not supported. If you have merged cells in your Excel file, it will be rejected by the platform.
